# MobileDevelopmentChallengeApp

A mobile app that allows users to browse movies, view characters associated with those movies, and read quotes from those characters. The app fetches movie data from a remote API and displays it in an easy-to-use interface.

## Features
- Movie List: Displays all movies with pagination.
- Character List: Shows all characters from a selected movie, also paginated.
- Character Details: On selecting a character, displays Name, Gender, Age (if available)
                                               A list of notable quotes from the character.
- Feature flag for enabling Quotes screen


## Technologies Used

- **Swift** – Main programming language for iOS development
- **SwiftUI** – For creating the user interface
- **URLSession** – For making network requests
- **XCTest** – For unit and UI testing
- **MVVM Code Clean Architecture** - Project Architecture

## Installation

### Prerequisites:
- **Xcode**

### Steps to Install:
- Open the project in Xcode.
– Select a simulator or connected device.
– Run the app by pressing Cmd + R.

### Steps to Run Unit tests:
- Run the unit tests using Xcode:
– Open the project in Xcode.
– Press Cmd + U to run all tests.
- View the test results in the Test Navigator
